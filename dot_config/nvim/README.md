## Shortcut (忘れやすいものをまとめておく)

### Terminal mode

| keymap | action |
| ---- | ---- |
| `<C-\><C-n>` | terminal-normal mode に切り替える |
| `i` | terminal-job mode に切り替える |
| `{number}<C-\>` | 該当の番号のターミナルを開く |
| `<C-r>` | `u` の実行を元に戻す |
| `{number}<C-\>` | 該当の番号のターミナルを開く |
| `<C-O>` | 元のカーソル位置に戻る |
| `<C-I>` | 元のカーソル位置に進む |
